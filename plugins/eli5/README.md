# eli5

Re-explain Claude's most recent message or explanation in the simplest possible terms, as if explaining to a 5-year-old.

## Usage

- `/eli5` — re-explain Claude's last message.
- `/eli5 <topic or text>` — explain the given topic/text instead.

It also triggers when you say things like "explain that like I'm 5", "simpler", or "I don't get it".

## What it does

Takes the last substantive explanation in the conversation and rewrites it using everyday analogies, short sentences, and plain words, opening with a one-line "big idea" and closing with a "So basically..." recap. It keeps the core truth intact rather than over-simplifying into something wrong.
